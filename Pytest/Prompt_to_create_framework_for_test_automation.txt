## Configuration Management

Implement a robust Configuration Manager class.

Requirements:

- Load YAML files dynamically based on environment.
- Support environments: test, stage, prod.
- Allow environment selection via CLI:
  - pytest --env=test
- Use singleton pattern.
- Validate configuration with Pydantic.
- Expose typed properties.

Example usage:

python settings = ConfigManager() base_url = settings.api.base_url browser = settings.ui.browser

---

## Secret Management

Implement secure secret handling.

Requirements:

- Load secrets from .env.
- Never commit secrets to Git.
- Support CI/CD environment variables.
- Provide typed access via SecretManager.

Example:

python secret_manager = SecretManager() username = secret_manager.api_username password = secret_manager.api_password

## API Client

Create a reusable enterprise API client.

Features:

- Base URL from configuration.
- Authentication support.
- Automatic headers.
- Request/response logging.
- Retry strategy.
- Timeout handling.
- Response validation.
- Support GET, POST, PUT, PATCH, DELETE.
- Attach requests and responses to Allure.

Example:

python response = api_client.get("/users/1")
